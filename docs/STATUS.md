The `content/*.json` files remain as the seed source + offline fallback + git history.
- **Media by URL.** Images referenced by URL (currently Matt's Squarespace CDN + a few repo assets); video = **native MP4 upload** (`<video>`, loop/autoplay) **or** Vimeo embed. Media uploads land in the public `project-media` Storage bucket ‚Äî confirm its allowed MIME types + file-size limit permit video (Supabase default ~50 MB/file) before large MP4s.
